# Metaphysical

| ----------- | ------------------- | ------------------ | -------------------------------------------------------------- |
| Nothingness | `null`, `undefined` | `NaN`              |                                                                |
| Emptiness   |                     | `false`, `0`, `""` | `[]`, `{}`                                                     |
| Substance   |                     |                    | `true`, non-zero numbers, non-empty strings, non-empty objects |
